# trajopt

A lightweight drone flight trajectory optimizer.

Dependencies:
- [Epigraph](https://github.com/EmbersArc/Epigraph)
- [Eigen](https://eigen.tuxfamily.org/index.php?title=Main_Page)

Set up:
```
./setup.sh
```

Build:
```
make
```

Run:
```
./main
```

Visualize:
```
python3 script/viz.py
```
